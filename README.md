## 1. 项目简介
这是一个基于微信公众号的小功能。通过微信公众号每天自动推送当日的最新文章。
不同的用户，可以订阅不同的文章，也可以取消订阅。

## 2. 项目结构
1、xmol文献爬取
2、微信公众号发送文章
3、sqlite数据库存储用户信息
4、web页面实现用户自助订阅功能
